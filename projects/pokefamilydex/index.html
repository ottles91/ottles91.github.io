PokeFamilyDex

This Python script generates a clean, human-readable National Dex-ordered list of Pok√©mon grouped by evolutionary family. This is a passion project of mine. I've always wanted to organize my Pokemon Home boxes by Pokemon family, but as far as I can tell there is no resource out there that clealy organizes Pokemon by family. I wanted to create a tool that would allow me to do this, and I hope it will be useful to others as well. The script is designed to be run in a Python environment and outputs the list in a text format that can be easily copied into a document or spreadsheet.
